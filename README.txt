How to install the mod:
Step 1: copy + paste the folder "siegerkranz" into your mod folder.
https://hoi4.paradoxwikis.com/Mod_structure#Local_Mod

Step 2: copy + paste the .mod folder into your mod folder (instructions above)

Step 3: open the .mod folder and fix the directory so that it knows where the "siegerkranz" folder is.

Step 4: open the "siegerkranz" folder and find the second .mod file. Open the .mod file and change the directory to the .mod file in your mod folder.
